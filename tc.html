SMP Rules : Minecraft Server

Here you can find cool people (Just like you) which are gamer too.

Before Proceeding in the server, you must agree with these rules :


  • Be Nice (Again. So be double nice)
  • Please respect the builds of other players
  • Do not spam or place blocks on other's build like houses, farm etc.
  • A public storage house is available near castle for all players, where you can store extra items from your inventory, other players can freely use it.
  • If someone says something that annoys or offends you, report the message to the Support team and contact an @Admin if needed. Do not engage with them and start a flame war. Just walk away.
  • If someone wants to do live streaming in the server,they can freely do this :)

Incase you break any of these rules, you will be banned from server for 1 week.
